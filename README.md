# 🦸‍♂️ Classificador de Nível de Herói

Este repositório contém soluções para o desafio de classificar o nível de um herói com base na experiência (XP) acumulada. O projeto foi implementado em três linguagens: **Python**, **Java** e **JavaScript**.

---

## 🎯 Descrição do Desafio

O objetivo é criar uma aplicação que armazene o nome e a quantidade de experiência de um herói e, em seguida, classifique o herói em um nível específico com base na XP. As categorias de nível são:

- **⚔️ Ferro**: XP menor que 1.000
- **🥉 Bronze**: XP entre 1.001 e 2.000
- **🥈 Prata**: XP entre 2.001 e 5.000
- **🥇 Ouro**: XP entre 5.001 e 7.000
- **💎 Platina**: XP entre 7.001 e 8.000
- **🚀 Ascendente**: XP entre 8.001 e 9.000
- **👑 Imortal**: XP entre 9.001 e 10.000
- **🌟 Radiante**: XP maior ou igual a 10.001

Ao final, o programa exibe a seguinte mensagem:

O Herói de nome {nome} está no nível de {nivel}
